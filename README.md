# 📝 To-Do App

## How to Run This App on Your Machine

### Step 1: Clone the Repository
```bash
git clone https://github.com/fantozy/react-todo-quizz.git
```

### Step 2:  Navigate to the project
cd react-todo-quizz

### Step 3: install dependencies
npm install

### Step 4: Run the dev server
npm run dev
